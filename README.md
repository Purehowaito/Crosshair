#### A Windows desktop application developed in C++ that draws a crosshair on the screen by creating a transparent topmost window, helping you with visual alignment and target positioning in games.
The program does not involve process injection, memory modification, or the installation of system drivers. As a result, it may be obscured by the game when the game is running in exclusive fullscreen mode. Please use it only in compliance with the game operator’s terms of service and permitted anti-cheat policies.

#### 基于C++开发的Windows桌面应用程序，通过创建透明的顶层窗口在屏幕上绘制准心，以帮助您在游戏中进行视觉定位。
程序不涉及进程注入、内存修改或系统驱动安装，因此游戏在以独占全屏下运行时可能会被覆盖；使用时您需遵循游戏运营方的用户协议及反作弊策略的允许下运行。

#### C++で開発されたWindowsデスクトップアプリケーションで、透明な最前面ウィンドウを作成して画面上に照準を表示し、ゲーム内での視覚的な位置合わせを補助します。
本プログラムは、プロセス注入、メモリ改変、システムドライバーのインストールを行いません。そのため、ゲームが排他フルスクリーンで実行されている場合、表示がゲーム側に隠れることがあります。ご利用の際は、ゲーム運営元の利用規約およびアンチチート方針の範囲内でご使用ください。

#### C++로 개발된 Windows 데스크톱 애플리케이션으로, 투명한 최상위 창을 생성해 화면에 조준점을 그려 게임 내 시각적 위치 맞춤을 돕습니다.
이 프로그램은 프로세스 주입, 메모리 수정, 시스템 드라이버 설치를 포함하지 않습니다. 따라서 게임이 독점 전체 화면 모드로 실행되는 경우 게임 화면에 의해 가려질 수 있습니다. 사용 시에는 게임 운영사의 이용 약관과 안티치트 정책을 준수해 주시기 바랍니다.
